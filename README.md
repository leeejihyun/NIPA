# 2020 인공지능 문제해결 경진대회

- 주관: 정보통신산업진흥원 (NIPA)

## 예선

## 기간
* 예선: 2020년 10월 26일 ~ 2020년 10월 28일
* 본선: 2020년 10월 30일 ~ 2020년 11월 13일
